# Anuhya T — Business Analyst Portfolio

A high-performance, 3D-interactive digital portfolio for Anuhya T, a Business Analyst specializing in delivering KPI dashboards, financial reporting analytics, and requirements-driven solutions.

## ✨ Features
- **3D Interactive Design**: Premium user experience with framer-motion animations and custom 3D transforms.
- **Dynamic Content Integration**: Dynamically bound to resume data for a personalized touch.
- **Responsive Layout**: Designed for seamless accessibility across all devices.
- **Modern Typography**: Utilizing Sora and Manrope font stack for a premium aesthetic.

Developed with React, TypeScript, Tailwind CSS, and Framer Motion.
